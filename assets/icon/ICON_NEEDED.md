# App Icon Required

SlabHaul needs a custom app icon.

## Specifications

- **Size**: 1024x1024px (source)
- **Format**: PNG with transparency
- **Style**: Should evoke crappie fishing
- **Colors**: 
  - Background: #0F172A (dark slate)
  - Accent: #0D9488 (teal)

## Suggested Concepts

1. Stylized crappie fish silhouette
2. Fish hook with "SH" letters
3. Fishing line forming a fish shape
4. Crappie with depth lines

## Files Needed

- `app_icon.png` - Main icon (1024x1024)
- `app_icon_foreground.png` - Android adaptive foreground

## Generation Command

Once icon files are in place:
```bash
flutter pub run flutter_launcher_icons
```
